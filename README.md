# CommitAI - AI-Powered Git Commit Messages

CommitAI is a micro-SaaS web application designed to help developers generate concise and relevant Git commit messages based on their uncommitted code or diffs. Simply paste your changes, and let AI suggest the perfect commit message.

Free tier available, with a one-time purchase option to unlock unlimited generations for a month.

## Features

- AI-powered commit message generation.
- Simple, intuitive web interface.
- Freemium model.
